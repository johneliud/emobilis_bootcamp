# EMobilis Bootcamp

## Introduction
This repository contains the assignment solutions and resources for the bootcamp.

## Getting Started

### Prerequisites
- Git

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/johneliud/emobilis-bootcamp.git
   ```
   
2. Navigate to the project directory:
   ```bash
   cd emobilis-bootcamp
   ```
